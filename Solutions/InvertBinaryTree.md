# Invert Binary Tree

---

## Interviewer Prompt

If you're given a binary tree, invert it!

```
Example:

Input:

     4
   /   \
  2     7
 / \   / \
1   3 6   9

Output:

     4
   /   \
  7     2
 / \   / \
9   6 3   1
```
---

## Interviewer Guide
